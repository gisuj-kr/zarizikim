# 자리지킴 리소스 디렉토리

이 디렉토리에는 앱에서 사용하는 아이콘 파일이 포함됩니다.

## 아이콘 파일
- `icon.png` - 기본 앱 아이콘 (256x256)
- `icon.ico` - Windows용 아이콘
- `icon.icns` - macOS용 아이콘
- `icon-idle.png` - 트레이 아이콘: 미출근 (16x16, 회색)
- `icon-working.png` - 트레이 아이콘: 근무중 (16x16, 녹색)
- `icon-away.png` - 트레이 아이콘: 자리비움 (16x16, 노란색)

## 참고
아이콘 파일이 없으면 앱에서 기본 아이콘을 자동 생성합니다.
나중에 커스텀 아이콘으로 교체할 수 있습니다.
